# 早报分析方法论指南

> **版本**: v1.3 | **最后更新**: 2026-06-14
> **定位**: A股早报生成的核心分析框架。主文件只保留规则、流程和决策标准；模板、样例表述、展开格式见 `stock_morning_brief_templates.md`。
> **原则**: 脚本管 FORMAT，LLM 管 INSIGHT；分析要可执行、可追踪、可迭代。

---

## 0. 核心原则

1. **先判断风险，再寻找机会**：早报优先识别隔夜风险、今日事件和仓位边界。
2. **先市场，后板块，再个股**：不在市场环境不支持时强行推荐方向。
3. **结论必须可执行**：每个判断都要落到仓位、节奏、方向或风险纪律。
4. **数据与观点分离**：数据来自脚本和行情源，观点来自方法论推理。
5. **择优迭代**：新样例更实用则吸收，旧规则不适用则降级或废弃。

## 2. 早报分析优先级

每日早报按以下顺序思考，避免泛泛而谈：

1. **隔夜外部风险**：美股、VIX、汇率、大宗、海外突发事件。
2. **昨日A股状态**：指数、成交量、涨跌家数、主线强度、资金流向。
3. **今日关键事件**：数据、会议、政策、产业催化、公司事件。
4. **市场阶段判断**：反弹、右侧、左侧、震荡、下跌。
5. **板块持续性**：主线是否延续，是否出现跷跷板或风格切换。
6. **操作策略**：仓位、节奏、低吸/观望/右侧确认。
7. **个股机会**：只在方向和节奏支持时给出。

## 3. 市场定位方法论

### 3.1 判断维度

| 维度 | 观察对象 | 结论落点 |
|---|---|---|
| 趋势 | 上证、创业板、科创50、均线系统 | 趋势向上/震荡/向下 |
| 位置 | 关键支撑压力、历史分位、前高前低 | 高位/中位/低位，风险收益比 |
| 阶段 | 反弹、修复、分化、加速、补跌 | 今日定性 |
| 节奏 | 连续阳阴线、量价、分时强弱 | 追随/等待/低吸 |

### 3.2 阶段定性

| 定性 | 含义 | 操作 |
|---|---|---|
| 反弹 | 下跌后的技术修复，趋势未确认 | 谨慎参与，不追高 |
| 右侧 | 趋势和资金确认 | 可提高参与度 |
| 左侧 | 提前试探，确定性不足 | 小仓试错 |
| 震荡 | 方向不明，板块轮动快 | 控仓，高抛低吸 |
| 下跌 | 风险释放未结束 | 防守，等待信号 |

## 4. 信号判断方法论

### 4.1 核心信号

| 信号 | 判断重点 | 用途 |
|---|---|---|
| 补跌完成 | 前期强势方向是否补跌 | 判断风险释放 |
| 量能变化 | 放量突破、缩量调整、地量 | 判断承接和活跃度 |
| 技术支撑 | MA60/MA120/前低/平台位 | 判断防守边界 |
| 情绪温度 | 涨跌家数、涨停跌停、连板高度 | 判断赚钱效应 |
| 资金回流 | 北向、主力、成交额 | 判断参与意愿 |
| 主线持续 | 龙头强度、板块扩散、分歧修复 | 判断方向可靠性 |

### 4.2 信号数量与仓位

| 满足信号 | 市场含义 | 仓位倾向 |
|---|---|---|
| 0-1项 | 信号不足 | 0-2成，防守观望 |
| 2-3项 | 有试探价值 | 1-3成，小仓试错 |
| 4项以上 | 信号较充分 | 3-6成，结构性参与 |

## 5. 情绪温度分类（P0）

| 情绪状态 | 市场特征 | 仓位建议 | `SENTIMENT_CLASS` |
|---|---|---:|---|
| 高涨 | 普涨、放量、主线强、涨停扩散 | 5-7成 | `sentiment-hot` |
| 温和 | 结构性机会，赚钱效应一般 | 3-5成 | `sentiment-warm` |
| 寒冷 | 跌多涨少，资金谨慎，主线弱化 | 1-3成 | `sentiment-cold` |
| 极寒 | 恐慌杀跌，跌停增多，流动性差 | 0-2成 | `sentiment-frozen` |

情绪判断必须服务于仓位建议和心态管理，不能只做形容词描述。

## 6. 事件时间线方法论（P0）

### 6.1 展示规则

1. **按时间排序**：已发生 → 今日 → 未来。
2. **最多10条**：只保留对A股风险偏好、流动性、主线板块、重要个股有影响的事件。
3. **已发生事件必须给结论**：一句话说明结果、落地情况或市场反馈。
4. **未发生事件必须给关注点**：说明看什么变量、可能影响什么方向。
5. **标签必须明确**：催化、利好、偏多、谨慎等。

### 6.2 事件优先级

| 优先级 | 事件类型 | 说明 |
|---|---|---|
| P0 | 全球宏观、央行、汇率、大宗突发 | 直接影响风险偏好和流动性 |
| P0 | 国内政策、监管、重要会议 | 影响A股方向和风险偏好 |
| P1 | 产业催化、价格变化、订单事件 | 影响主线板块持续性 |
| P1 | 龙头公司事件 | 影响产业链映射和情绪 |
| P2 | 普通公司新闻、弱相关题材 | 仅在无更重要事件时保留 |

### 6.3 输出要求

- 已发生：`日期 + 标签 + 事件 + 一句话结果/结论`
- 今日/未来：`日期 + 标签 + 事件 + 关注变量/潜在影响`
- 禁止堆砌新闻标题，必须转化成交易影响。

## 7. 逻辑挖掘方法论

### 7.1 逻辑分类优先级

| 逻辑类型 | 持续性 | 判断标准 | 参与倾向 |
|---|---|---|---|
| 涨价逻辑 | 高 | 价格上涨可传导到业绩 | 优先 |
| 替代逻辑 | 高 | 国产替代、供应链重构、订单验证 | 优先 |
| 业绩线 | 高 | 业绩预增、订单增长、指引上修 | 优先 |
| 事件驱动 | 中 | 政策、会议、IPO、产业事件 | 看持续性 |
| 红利板块 | 中 | 高股息、低波动、防守属性 | 震荡防守 |
| 情绪修复 | 低 | 超跌反弹、短期修复 | 谨慎 |
| 题材炒作 | 极低 | 只有故事缺少数据 | 回避 |

### 7.2 板块方向输出规则

每个方向必须包含：

1. 方向名称
2. 核心催化
3. 持续性判断
4. 参与方式
5. 风险点

模板见 `stock_morning_brief_templates.md`。

## 8. 主题催化与产业链映射方法论

### 8.1 学习素材处理边界

用户提供的盘面、消息、行业材料需先判定用途：

| 素材类型 | 处理方式 |
|---|---|
| 方法论学习素材 | 只抽取分析方法、判断框架、产业链映射逻辑，不直接写入报告 |
| 当日报告输入 | WebSearch验证后再进入事件、板块、策略和选股 |
| 选股素材 | 仅涉及具体股票筛选、评分、避坑的内容进入 `stock_selection_guide.md` |

学习素材必须区分“方法”与“事实”：事实数据未经验证不得成为报告结论。

### 8.2 新概念权威背书识别

当传统行业被重新命名，并与当前最强主线绑定，同时获得权威媒体、政策口径、产业资本或交易产品背书时，应提高其主题扩散权重。

```text
旧行业/传统资产 → 新概念命名 → 权威背书 → 市场共识形成 → 产业链重估
```

判断重点：

1. 新概念是否绑定当前主线，而不是孤立造词。
2. 背书来源是否具备传播和定价影响力。
3. 是否能解释上游、设备、材料、能源、连接等环节的重估。
4. 是否有资金承接或板块扩散迹象。

### 8.3 强主线上游重估法

分析强主线时，不只看最显性的设备、应用和龙头，还要沿产业链向上游和基础设施反推，寻找尚未充分定价的低位补涨方向。

```text
终端需求增长 → 基建/设备扩张 → 材料/能源/连接需求上升 → 上游资源或基础设施重估
```

适用场景：AI算力、新能源、低空经济、机器人、商业航天等中长期主线。

输出要求：早报中应写清“终端需求如何传导到上游”，不能只写概念名。

### 8.4 排产验证景气法

产业链景气判断不能只看价格和新闻，也要看排产、订单和淡旺季特征。

```text
排产同比高增 + 环比改善 + 淡季不淡 + 下游订单能见度提升 = 景气改善前置信号
```

判断标准：

| 信号 | 含义 |
|---|---|
| 同比高增 | 行业需求较去年明显改善 |
| 环比改善 | 短期趋势仍在向上 |
| 淡季不淡 | 需求强于季节性规律 |
| 订单能见度延长 | 从短期交易逻辑升级为中期景气重估 |

该方法用于盘前板块判断和策略，不直接等同于个股买入信号。

### 8.5 海外资金工具映射

海外市场围绕中国核心资产设计杠杆ETF、单票ETF或其他交易工具时，说明相关资产可能具备全球资金定价属性。

```text
海外资金工具出现 → 资金审美变化 → 全球资产定价属性提升 → A股同产业链映射
```

分析时关注“资金工具背后的审美”，而不是简单把ETF申报等同于利好。需观察是否能映射到A股产业链、龙头估值和主题扩散。

### 8.6 AI主线“存、算、连”拆解

AI产业链分析应拆为三条核心链路，并继续向上游基础环节延伸：

| 环节 | 观察方向 | 二级映射 |
|---|---|---|
| 存 | 存储、HBM、DRAM、SSD、服务器存储 | 存储芯片、材料、封装测试 |
| 算 | GPU、ASIC、AI服务器、液冷、电力 | 电力设备、铜铝钼等材料、散热 |
| 连 | 光模块、CPO、交换机、连接器、光芯片 | 光通信、PCB、高速连接、设备材料 |

使用该框架时，早报应判断当前资金炒作的是“存、算、连”哪一环，以及是否出现向上游或低位分支扩散。

## 9. 外部关联方法论

### 9.1 外部因素分类

| 类型 | 观察重点 | A股映射 |
|---|---|---|
| 美股走势 | 三大指数、纳指、VIX、科技股 | 风险偏好、科技映射 |
| 汇率与美元 | 美元指数、离岸人民币 | 外资、成长股估值 |
| 大宗商品 | 原油、黄金、铜等 | 通胀、资源、避险 |
| 海外公司事件 | 订单、合作、业绩、产能 | 产业链映射 |
| 宏观数据 | CPI、PMI、非农、议息 | 流动性预期 |
| 重大突发 | 地缘、制裁、金融风险 | 风险偏好冲击 |

### 9.2 分析路径

```text
事件/数据 → 影响风险偏好/流动性/产业链 → 对A股指数和板块的影响 → 今日应对策略
```

## 10. 操作策略方法论

### 10.1 三段式输出

1. **今日仓位**：给出建议区间，并说明相比昨日是否加减仓。
2. **参与节奏**：低吸、观望、右侧确认、逢分歧布局、不追高。
3. **风险纪律**：明确什么情况减仓、停止开仓或等待确认。

### 10.2 仓位参考

| 市场阶段 | 仓位参考 | 操作节奏 |
|---|---:|---|
| 下跌中 | 0-2成 | 防守等待 |
| 见底中 | 1-3成 | 小仓试探 |
| 震荡期 | 3-5成 | 高抛低吸 |
| 反弹中 | 5-7成 | 等分歧，控节奏 |
| 主升中 | 7成以上 | 顺势持有，动态止盈 |

## 11. 风险预警方法论

| 风险类型 | 观察指标 | 应对 |
|---|---|---|
| 外部风险 | 美股大跌、VIX上升、汇率贬值 | 降仓或延后开仓 |
| 流动性风险 | 缩量、资金流出 | 谨慎参与 |
| 情绪风险 | 连板断裂、跌停增加 | 回避高位股 |
| 主线风险 | 核心板块补跌 | 等分歧结束 |
| 事件风险 | 数据/会议未落地 | 等结果再行动 |

风险预警必须写清楚“触发条件”和“应对动作”。

## 12. 择优录取机制

收到新样例时，按以下流程更新方法论：

1. 提取样例的市场定位、信号判断、逻辑挖掘、外部关联、操作策略。
2. 与现有方法论比较：可操作性、可量化性、实用性、全面性、时效性。
3. 样例更优则吸收，AI更精确则保留，二者互补则融合。
4. 更新版本号，并记录来源、变更内容、替代了什么旧方法。
