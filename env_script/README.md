# Node.js 环境检测与安装脚本

Windows 环境下自动检测和安装 Node.js 的脚本工具。

## 脚本说明

### 1. install_nodejs.bat (批处理脚本)

**特点：**
- 兼容性好，支持 Windows 7 及以上版本
- 无需额外权限设置
- 自动检测系统架构 (x86/x64)
- 支持断点续传检测

**使用方法：**
```cmd
# 方法 1: 双击运行
双击 install_nodejs.bat 文件

# 方法 2: 命令行运行
cd env_script
install_nodejs.bat
```

### 2. install_nodejs.ps1 (PowerShell 脚本)

**特点：**
- 界面更美观，带进度显示
- 更详细的错误处理
- 下载进度可视化
- 彩色输出

**使用方法：**
```powershell
# 方法 1: 右键运行
右键点击文件 -> 使用 PowerShell 运行

# 方法 2: PowerShell 命令行
cd env_script
powershell -ExecutionPolicy Bypass -File install_nodejs.ps1

# 方法 3: 以管理员身份运行 (推荐)
右键点击 PowerShell -> 以管理员身份运行
cd env_script
.\install_nodejs.ps1
```

### 安装详情

- **Node.js 版本**: v20.15.0 (LTS 长期支持版)
- **安装方式**: MSI 安装包 (静默安装)
- **安装路径**: 默认 `C:\Program Files\nodejs`
- **包含工具**: Node.js + npm

## 系统要求

- Windows 7 SP1 或更高版本
- 管理员权限 (安装时需要)
- 网络连接 (下载安装包)

## 常见问题

### Q1: 提示"无法识别 node 命令"

**解决方案：**
1. 关闭当前命令行窗口
2. 重新打开一个新的命令行窗口
3. 运行 `node -v` 检查

### Q2: 下载速度很慢

**解决方案：**
- 使用国内镜像下载
- 手动从 https://nodejs.org/ 下载

### Q3: 安装失败

**解决方案：**
1. 确保以管理员身份运行
2. 临时关闭杀毒软件
3. 检查网络连接
4. 手动下载 MSI 安装包

## 手动安装

如果脚本无法正常工作，可以手动安装：

1. 访问 Node.js 官网: https://nodejs.org/
2. 下载 LTS 版本 (长期支持版)
3. 运行安装程序
4. 按照向导完成安装

## 验证安装

安装完成后，打开新的命令行窗口，运行：

```cmd
node -v
# 应显示: v20.15.0

npm -v
# 应显示类似: 10.7.0
```
